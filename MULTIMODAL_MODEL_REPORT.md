## 📝 使用说明

### 语音输入
1. 点击输入框左侧的麦克风图标
2. 允许浏览器访问麦克风
3. 开始说话，语音会实时转换为文字
4. 再次点击麦克风图标停止录音

### Prompt 模板
1. 点击输入框左侧的模板图标
2. 选择合适的模板分类
3. 点击模板名称应用模板
4. 根据需要修改模板内容

### 数学公式
- 支持使用 Markdown 数学公式语法
- 例如：`$E=mc^2$` 或 `$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$`

## 🔧 技术细节

### 语音识别 API
- 使用 `webkitSpeechRecognition` 或 `SpeechRecognition`
- 支持语言：`zh-CN`（中文）
- 连续模式：关闭
- 实时结果：开启

### Prompt 模板
- 存储在 JavaScript 数组中
- 支持变量替换（如 `{主题}`、`{代码}`）
- 自动填充到输入框

### 输入框
- 最大高度：200px
- 最小高度：52px
- 自动调整：基于内容高度
